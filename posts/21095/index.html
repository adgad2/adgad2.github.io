HTML+CSS练习之百度首页

发表于 2017-05-18   |  

稍微学了下html+css，决定做个小练习，本来想做google，发现即使是外面像也不容易，于是选择相对简单的百度。
先上百度原图：

然后我仿制的网页：

链接：http://mrone.top/%E7%99%BE%E5%BA%A6.html
是学完一段时间后捡回来学的，感觉都快丢光了，加油吧，啊啊啊，好想换工作。真正制作才会发现学和做是两回事，首先是细节，多翻手册，然后，重要的是制作时善用google,很多问题都已有大牛解决。

贴上代码吧，百度简单地分为3个部分，头部，中间logo和搜索框，底下的版权信息。
html：

<div id="top">
    <div>
        <a href="#1">新闻</a>
        <a href="#">hao123</a>
        <a href="#">地图</a>
        <a href="#">视频</a>
        <a href="#">贴吧</a>
        <a href="#">学术</a>
        <a href="#" style="font-weight: normal;">XXXXXXXX</a>
        <a href="#" style="font-weight: normal;">设置</a>
        <a href="#" id="list" style="font-weight: normal;">更多产品</a>
    </div>
</div>

css部分写的有点乱：
*{margin:0;padding:0};
body{ text-align:center}
.div{ margin:0 auto; border:1px solid #F00}

    width: 100%;

    float: right;
    text-align: right;
}

    font-weight: bold;
    color: #555;
    font-size: 13px;
}

#top a{
    margin-left: 15px;
    line-height: 32px;
}

    font-weight: normal;
}

    float: right;
    height: 32px;
    width: 86px;

    color: white;

这里只用了一个内嵌套，主要是排班和颜色的调整，没什么难度、

logo和搜索框部分的html：

<hr color="#EEEEEEEE;" size="1px;" />
<div id="logo"><img src="./bd_logo.png" alt="" width="270" height="129"></div>

<form action="form_action.asp">
    <span><input id='search' type="text" name="search"></span><span><input id="button" type="button" value="百度一下"></span>
</form>

这里用的是一个表单，最上面是分隔线，略微调整一下color和size即可。
css比较简单，主要是框和long大小、位置的调整：

    #logo{
        text-align: center;
        margin-top: 73px;
        width: 100%;
    }

    form{
        text-align: center;
        margin-top: 18px;
        margin-left: 390px;
    }
    #search{
        width: 536px;
        height: 38px;
        border-color: ;
    }

        width: 102px;
        height: 42px;
        border: 0px;

        background: #38f;
        margin: 0px;
        font-size: 17px;
        color: white;
    }
span{
    float: left;display: inline-block;height: 40px;
}

框和搜索按钮无法对齐，Google后用一个span包住解决。
然后是底部fotter：

有点乱哈，主要是内嵌了部分css和图片网址，难点在于对齐，包括文字与文字的对齐，文字与图片的对齐,其它就是文字的微调。
css:

    vertical-align:middle;
    position: absolute;
    width: 100%;

    text-align: center;
}

    vertical-align:middle;
    font-size: 12px;
    color: #999;
    line-height: 16px;
