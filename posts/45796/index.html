管理文章里包括删除和修改。
管理文章页面：

说一下删除功能的实现，先上代码。

<?php 
require_once('../connect.php');//连接文件，引入这个文件，已经成功连接到数据库
$id=$_GET['id'];//得到要删除的文章ID
$deletdsql="delete from article where id = $id"; //通过sql语句进行删除

if( mysqli_query($con,$deletdsql)){
    echo "<script>alert('删除成功');window.location.href='article.manage.php</script>";
}else{
    echo "<script>alert('删除失败');window.location.href='article.manage.php</script>";
}
 ?>

删除功能比较简单。

然后是修改功能，先上代码。

<?php
require_once('../connect.php');
$id = $_GET['id'];//得到旧文章的ID
$query = mysqli_query($con,"select * from article where id=$id");//通过ID获取到这个资源
$data = mysqli_fetch_assoc($query);//打开资源
?>

之后在通过post模式将表单发送到另一个php，代码如下：

<?php 
require_once('../connect.php');

$id = $_POST['id'];//以下都是获取post传递进来的需要修改的数据。
$title = $_POST['title'];
$author = $_POST['author'];
$description = $_POST['description'];
$content = $_POST['content'];
$dateline =  time();
$updatesql = "update article set title='$title',author='$author',description='$description',content='$content',dateline=$dateline where id=$id";//用update语句进行更新
if(mysqli_query($con,$updatesql)){//显示成功或不成功，并返回到管理页面
    echo "<script>alert('修改文章成功');window.location.href='article.manage.php';</script>";
}else{
    echo "<script>alert('修改文章失败');window.location.href='article.manage.php';</script>";
}
?>

发布的文章的和修改类似，也是通过post传递，然后使用insert语句加入到数据库里。

之后是文章的前台页面：
：
首先是前台php代码的实现：

<?php 
require_once('connect.php');
$sql = "select * from article order by dateline desc"; //将文章根据时间降序取出
$query = mysqli_query($con,$sql);
if($query&&mysqli_num_rows($query)){//当$query语句执行正确并且数据不为0时输出文章
    while($row = mysqli_fetch_assoc($query)){
        $data[] = $row;//将数据放入数组中，成为一个二维数组
    }
}
?>
