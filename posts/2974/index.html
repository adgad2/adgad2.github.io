去掉PHPstorm在写原生SQL的黄色波浪线并增加提示

发表于 2018-09-07 | 分类于 技术向

在PHPstorm写原生SQL的时候，一直有黄色波浪线的提示，如下图，今天就想着解决一下。

提示语如下 “No data sources are configured to run this SQL and provide code assistance.Disable this inspection via problem menu”， 既然是有问题，我怎么就简单地关闭这个检查呢。

经过Google，我在PHPstorm官网找到了解决方案，原文连接如下：

https://www.jetbrains.com/help/phpstorm/managing-data-sources.html

按照文章的指示，我们可以打开视图（view）- 工具窗口（Tool Windows）-database，或者在右侧直接打开。

点击加号，database source-mysql，选择项目使用的，我这里选择的是mysql，

然后按照提示填入相应信息，注意，第一次使用，Test Connection按钮可能是灰的，在红色圈中会叫你安装一些相应的驱动，安装即可，全自动。

最后Test Connection一下，如果成功会显示 successful，点击确定

这个时候我们回去看那条SQL，黄色波浪线消失了，字段和表颜色也表了，而且有了提示，如下图：

至此，大功告成，烦人的波浪线没了，还增加了提示，是不是美滋滋？
